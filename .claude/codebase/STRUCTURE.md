## Entry Points (exact)
- Operator: `cmd/redshiftsink/main.go:72` `func main` → builds Manager, registers `RedshiftSinkReconciler` (`:129`), `mgr.Start` (`:159`).
- Reconcile loop: `controllers/redshiftsink_controller.go:653` `Reconcile` → `:329` private `reconcile`.
- Controller wiring: `controllers/redshiftsink_controller.go:724` `SetupWithManager`.
- Batcher: `cmd/redshiftbatcher/main.go:160` `func main` → `:51` `run` (cobra).
- Loader: `cmd/redshiftloader/main.go` `func main` → `run` (cobra).
- Batcher data path: `pkg/redshiftbatcher/batch_processor.go:498` `Process`.
- Loader data path: `pkg/redshiftloader/load_processor.go:835` `Process`.

## Module Boundaries & Dependency Direction
- `cmd/*` → `controllers/` (operator) and `pkg/*` (workers).
- `controllers/` → `api/v1`, `pkg/git`, `pkg/kafka`, `pkg/prometheus`, `pkg/redshift`, `pkg/notify`.
- `pkg/*` are leaf libraries; `pkg/redshiftbatcher` and `pkg/redshiftloader` depend on `pkg/kafka`, `pkg/serializer`, `pkg/transformer`, `pkg/s3sink`, `pkg/redshift`, `pkg/schemaregistry`.
- `api/v1` depends only on k8s apimachinery — the stable contract everything else imports as `tipocav1`.

## Configuration File Locations
- Operator config: command-line flags only (`cmd/redshiftsink/main.go` `flag.*Var`), no config file.
- Worker config: YAML loaded by Viper — `cmd/redshiftbatcher/config/config.go` + `config_sample.yaml` (default path `./cmd/redshiftbatcher/config/config.yaml`), same for loader. Real `config.yaml` is git-ignored (`cmd/*/config/.gitignore`).
- CR samples: `config/samples/`.
- Secret sample: `config/operator/kustomization_sample.yaml`.

## Build Output
- Binaries: `bin/<os>_<arch>/<binary>` (e.g. `bin/linux_amd64/redshiftbatcher`). Intermediate build state under `.go/` (git-ignored).
- Images: `public.ecr.aws/practo/<binary>:<version>`.

## Generated vs Hand-Written
- **Generated** (do not hand-edit): `api/v1/zz_generated.deepcopy.go` (controller-gen `make generate`), `config/crd/bases/*` (controller-gen `make manifests`).
- **Hand-written**: everything else. Editing `api/v1/redshiftsink_types.go` requires re-running `make generate manifests`.
